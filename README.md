📝 Notepad App
A lightweight and feature-rich Notepad App built using Kotlin for Android, designed for quick and easy note-taking.

✨ Features
✅ Create & Edit Notes – Jot down ideas instantly
✅ Save & Delete Notes – Manage your notes easily
